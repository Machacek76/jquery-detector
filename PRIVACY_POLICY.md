# Privacy Policy for jQuery Version Detector

**Last Updated:** November 21, 2025

## Overview

jQuery Version Detector is a browser extension designed to detect and display jQuery library versions on web pages. This privacy policy explains how the extension handles data.

## Data Collection

**We do NOT collect, store, or transmit any user data.**

Specifically:
- ❌ No personal information is collected
- ❌ No browsing history is recorded
- ❌ No user activity is tracked
- ❌ No cookies are used
- ❌ No analytics or telemetry
- ❌ No data is sent to external servers

## What the Extension Does

The extension:
- ✅ Reads page content (script tags and JavaScript objects) to detect jQuery
- ✅ Displays detection results locally in your browser
- ✅ Changes icon colors based on jQuery version found
- ✅ All processing happens locally on your device

## Permissions Used

### Host Permissions (http://*/* and https://*/*)
- **Purpose:** To scan web pages for jQuery library
- **Usage:** Only reads page resources (script tags, window objects)
- **Data Storage:** None - all processing is done in real-time

### activeTab
- **Purpose:** To detect jQuery on the currently active tab
- **Usage:** Triggered only when you click the extension icon
- **Data Storage:** None

### scripting
- **Purpose:** To inject detection code into web pages
- **Usage:** Executes only our detection function, no external code
- **Data Storage:** None

### tabs
- **Purpose:** To manage extension icons per tab
- **Usage:** Only to identify tabs and set appropriate icons
- **Data Storage:** None

### storage
- **Purpose:** Reserved for future user preferences (e.g., language)
- **Usage:** Currently minimal or unused
- **Data Storage:** Only local preferences, no personal data

## Third-Party Services

**None.** This extension does not use any third-party services, analytics, or external connections.

## Data Sharing

**We do not share any data** because we do not collect any data.
